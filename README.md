# 🏔️ Calculadora de Materiales - Comercial de la Patagonia

Herramienta técnica profesional diseñada para la cubicación precisa de materiales de construcción, optimizada para las necesidades específicas de la **Región de Magallanes y la Antártica Chilena**.

## 🚀 Características Principales

- **Cubicación Técnica**: Cálculos precisos para cemento, ladrillos, mallas, techumbre y más.
- **Diseño Fine Sans**: Interfaz de alta gama, minimalista y corporativa.
- **Módulo de Cotización**: Gestión de materiales en tiempo real con capacidad de edición inteligente.
- **Optimización Móvil**: Experiencia fluida y adaptable para uso en terreno.
- **Conexión Directa**: Integración con WhatsApp oficial y canales de atención de Comercial de la Patagonia.

## 🛠️ Tecnologías Utilizadas

- **React + TypeScript**: Para una lógica de cálculo robusta y tipada.
- **Tailwind CSS**: Diseño responsivo y estética de lujo.
- **Lucide React**: Iconografía técnica nítida.

## 📍 Ubicación e Identidad

Desarrollado para **Comercial de la Patagonia**, con base en la Zona Franca de Punta Arenas, Chile.
